# Webpack Boilerplate

## Features

-   [x] JS Bundle
-   [x] Typescript Support
-   [x] Babel
-   [x] minify Javascript,HTML,CSS
-   [x] SASS/CSS compile
-   [x] static directory
-   [x] dev server
-   [x] ESLint

## Build Setup

```bash
# install dependencies
$ yarn install

# start dev
$ yarn dev

# production build
$ yarn build

```
